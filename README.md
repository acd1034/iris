# Light Concept Library for C++17
supported metafunctions:

`<concepts>`
- `is_derived_from`
- `is_common_reference`
- `is_common`
- `is_boolean`
- `is_weakly_comparable_with`
- `is_equality_comparable`
- `is_equality_comparable_with`
- `is_totally_ordered`
- `is_totally_ordered_with`
- `is_movable`
- `is_copyable`
- `is_semiregular`
- `is_regular`

`<iterator>`
- `is_iterator`
- `is_input_iterator`
- `is_output_iterator`
- `is_forward_iterator`
- `is_bidirectional_iterator`
- `is_random_access_iterator`

`<ranges>`
- `is_range`
